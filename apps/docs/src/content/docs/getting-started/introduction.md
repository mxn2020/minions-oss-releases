# Minions Oss-releases

Release definitions, changelogs, and publish receipts

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-oss-releases/sdk`) — Core library for Node.js
- **Python SDK** (`minions-oss-releases`) — Python bindings
- **CLI Tool** (`@minions-oss-releases/cli`) — Command-line interface
- **Web App** — Interactive playground at [oss-releases.minions.wtf](https://oss-releases.minions.wtf)
