---
title: Installation
description: Install Minions Oss-releases packages
---

## TypeScript / Node.js

```bash
npm install @minions-oss-releases/sdk minions-sdk
```

## Python

```bash
pip install minions-oss-releases
```

## CLI

```bash
npm install -g @minions-oss-releases/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
