# Welcome to Minions Oss-releases

We're excited to introduce **Minions Oss-releases** — Release definitions, changelogs, and publish receipts.

## What's Included

- **TypeScript SDK**: `npm install @minions-oss-releases/sdk`
- **Python SDK**: `pip install minions-oss-releases`
- **CLI**: `npm install -g @minions-oss-releases/cli`

## Getting Started

Check out the [documentation](https://oss-releases.minions.help) for a full guide.
